## 👋 About

> **외주 웹앱과 개인 SaaS를 혼자 만들어 출시하는 풀스택 개발자.**
> Next.js · FastAPI · Spring Boot · Kotlin Multiplatform으로 프론트·백·모바일·배포까지 한 사람이 책임지고,
> LLM을 장난감이 아니라 **제품**으로 붙입니다. 그리고 Claude Code 하네스를 손으로 깎습니다.

**💼 제공 가능한 서비스**

| 카테고리 | 내용 |
|---------|------|
| 🌐 **풀스택 웹 개발** | Next.js 16 · React 19 · FastAPI · Spring Boot · Kotlin — 프론트부터 DB·배포까지 한 사람이 끝까지 책임 |
| 🤖 **AI 제품 엔지니어링** | LLM 오케스트레이션(OpenRouter · Spring AI · LangGraph) · Structured Output · SSE 스트리밍 · 파일 파싱 |
| 📱 **Kotlin Multiplatform 모바일** | Compose Multiplatform · Clean Architecture · 한 코드베이스로 iOS + Android 동시 빌드 |
| ⚙️ **이중 AI 하네스 운영** | Everything Claude Code(ECC) 16 플러그인 · 28 에이전트 · 115 스킬 · 59 커맨드 + OpenClaw 자체 봇 |

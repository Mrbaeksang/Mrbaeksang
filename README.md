## 🪝 정확한 AI 활용 — 매일 쓰는 4종 훅

**모델이 빠르고 똑똑해도, 호출 한 번에 30만 토큰을 흘리거나 컨텍스트 윈도우가 raw 출력으로 가득 차면 답은 둔해집니다.** 토큰 · 컨텍스트 · 맥락이 새지 않게 매일 쓰는 훅 네 가지.

| 훅 | 어떤 손실을 막나 |
|----|------------------|
| **RTK** _(Rust Token Killer)_ | Claude Code 훅에 자동 wrapping된 CLI proxy. `git status` · `npm install` 같은 일상 명령 출력을 모델 컨텍스트로 흘리기 전에 요약·필터링해서 **토큰 60~90% 절감** |
| **context-mode** | Bash 출력 · WebFetch HTML · 큰 JSON을 메모리에 흘리지 않고 **샌드박스 + FTS5 인덱스**로 받음. `ctx_search`로 필요한 부분만 꺼내서 컨텍스트 윈도우 잠식 방지. 14개 클라이언트 호환 |
| **graphify** | 코드·문서·이미지 폴더를 **지식 그래프**로 만들어 BFS · DFS · 최단 경로 쿼리로 답. "X가 어디서 호출돼?" 같은 질문에 raw 파일 안 읽고 그래프 traversal로 |
| **Atrium** _(철학)_ | 외부 API = 비용 · 검열 · 지연 · 프라이버시 4종 묶음. **로컬 GPU에 모델 직접 띄워 MCP로** Claude Code가 호출하면 그 묶음에서 풀려요. 매 워크플로마다 "어디까지 로컬로 가능한가"부터 봅니다 |

```
                       └─ 5s 안: 동기 / 5s 초과: callbackUrl 비동기
```

| 레이어 | 구성 |
|--------|------|
| **메시지 채널** | 카카오톡 채널 (사업자 백상랩스) + 카카오 i 오픈빌더 Skill 서버. 양방향 통신, callbackUrl로 긴 응답 비동기 |
| **에이전트 코어** | Hermes Agent (Nous Research / MIT). Persistent memory · Skills Hub · 자연어 cron 스케줄러 |
| **추론 엔진** | 로컬 Qwen3.6 35B-A3B Uncensored (OpenAI-compat `:8080`) + Qwen3-Embedding-8B (`:8082`) |
| **외부 노출** | Tailscale Funnel / Cloudflare Tunnel로 로컬 endpoint를 공인 도메인에 매핑 |
| **통합 스킬** | Gmail · Google Calendar · Linear · arXiv · 매일의 AI 발행 |
